# 2 Нормальная форма
## Правило

Правила 1НФ соблюдены, и каждый неключевой атрибут функционально полно зависит от первичного ключа.

Функциональная зависимость атрибута заключается в том, что для каждого атрибута A соответствует в точности одно значение B (A→B), это означает, что во всех кортежах с одинаковым значением атрибута A атрибут B будет иметь одно и то же значение.

# 3 Нормальная форма
## Правило

Правила 2НФ соблюдены и каждый неключевой атрибут не транзитивно зависит от первичного ключа.

Атрибут C зависит от A транзитивно если для атрибутов A, B и C выполняются условия: A→B; B→C; обратная связь отсутствует.
